# 🌱 Capstone Generator (English Only)

## Quick Start
```bash
echo "GROQ_API_KEY=gsk_..." > .env
./setup.sh
./start.sh
```
Open http://localhost:3000

## Deploy
```bash
docker build -t capstone .
docker run -p 5000:5000 -e GROQ_API_KEY=gsk_... capstone
```

## Features
- 🤖 AI content (Groq) · 🎨 15,000+ CV layouts
- 🌍 France, Belgium, Ghana, Ireland
- 📄 137 real universities · 📊 Preview before download
